A modern, responsive web application for storing, organizing, and sharing your college memories with beautiful animations and offline capabilities.

## Features ✨

- 🖼️ **Image Gallery** - Responsive grid layout for all your college photos
- 🔍 **Search & Filter** - Find images by title, tags, or category
- 📥 **Download & Share** - Individual downloads or share memories with friends
- ☁️ **Offline Access** - Progressive Web App (PWA) support for offline viewing
- 🎨 **Modern UI** - Glassmorphism design with smooth animations
- 🌓 **Theme Switcher** - Toggle between light and dark modes
- 📤 **Easy Upload** - Drag & drop or file selection for adding new photos
- 🗂️ **Categories** - Organize by Events, Friends, Fests, Hostel, Classes, etc.
- ⚡ **Fast Performance** - Optimized for quick loading and smooth interactions

## Technologies Used 🛠️

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Design**: Glassmorphism, Gradient Backgrounds, CSS Animations
- **Storage**: IndexedDB (client-side storage)
- **Offline**: Service Workers (PWA)
- **Icons**: Font Awesome
- **Build**: Pure vanilla JS (no frameworks)

## Installation & Setup ⚙️

1. Clone the repository:
   ```bash
   git clone https://github.com/Sudarshanbadli/Collegememo.git
   cd Collegememo.git
   ```

2. Serve the application (you'll need Node.js installed):
   ```bash
   npx serve
   ```

   Or simply open `index.html` in your browser.

3. For PWA installation:
   - In Chrome/Edge: Click "Install" when prompted or find the option in the browser menu
   - In Firefox: Add to Home Screen from the browser menu

## Usage Guide 📖

### Adding Photos
1. Click "Upload Photos" from the main menu
2. Drag & drop images or click to select files
3. Add title, category, and tags
4. Click "Upload Selected Files"

### Browsing Photos
- Use category tabs to filter images
- Search by title or tags in the search bar
- Click any image to open in lightbox view

### Lightbox Features
- ← → arrows to navigate between photos
- 🔍 Zoom in/out functionality
- ⬇️ Download individual photos
- ↗️ Share via native share dialog
- ⛶ Toggle fullscreen mode
